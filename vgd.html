Video Game Development

Since I first put my hands on a computer, video games have amazed me. The use of computer graphics, physics, and logic processing all rolled into one program to simulate real-world and fantasy-world functions, is what made me want to start learning about Computer Science from a very young age. I would love to create a videogame such as the masters before me, for the world to enjoy, and be able to represent myself on a canvas with no-limits.

Disco Panic - 2D Game

Being developed for a course CCT423 - Game Development Project. Disco Panic is a 2D game developed in Unity, utilizing .NET C# to create a wave spawning game, that you must eliminate enemies to progress, and as you do you gain more dance moves, can change music on a jukebox, and unlock doors. All the music, artwork, and sprites are animated, and created by me.
Byte Flight - 3D Android Mobile Game

A personal project begun in the summer of 2018. The world of three-dimensional programming had been intimidating to me, so naturally, I thought it was going to be a complex challenge, however, I found out that using C# and the Unity Development Environment, I could begin to understand and manipulate 3D space using OOL. Java being natural to me for over 7 years at this point, C# was easily picked up. I drew a design I wanted, an endless randomized tunnel flying game. By the end of the summer it had been completed, and I uploaded it on the Play Store. You can now download the APK from my GitHub.
Children's Educational Game

Built in high-school as a final project, we were tasked with using Visual Basic to program a children's educational game. I was inspired by my child-hood games I played and built a point and click adventure with animated characters in the background moving across the screen, and a math game that could be played with potions and effects everytime the answer was correct. My teacher was so impressed by it, he asked if he could keep my hand-drawn mock-ups of the game.

Created by Antonio LaPlaca with Bootstrap. Check out my GitHub.
